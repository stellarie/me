yomitori

personal library for 40,000+ scattered ebooks.

01 · meta

2026 · typescript · kotlin · rust · scss · docker

02 · what

placeholder. yomitori turns a folder of 40k+ ebooks into a searchable, beautiful library. crawls automatically, extracts covers and metadata, tracks reading progress, integrates with an epub reader. phase 2 added japanese-specific tools — furigana rendering, frequency-based vocab, srs quizzes, anki sync.

03 · why

placeholder. my ebooks lived in a folder named _ebooks. nothing was findable. wanted a real home — search with love, not find . -name.

04 · how

placeholder. typescript frontend, kotlin backend, rust for the heavier text-processing work (tokenization, frequency analysis). scss for styling. docker compose orchestrates the services. crawler runs on a schedule, extracts covers and metadata, persists to a local store. rest api drives search, pagination, and filtering.
