aozora viewer

a reader for japan's public-domain literary archive.

01 · meta

2026 · javascript · kotlin · html · css

02 · what

aozora bunko hosts thousands of out-of-copyright japanese literary works in raw xhtml. this project scrapes the archive into a clean catalog and renders the texts in a modern reader — japanese typography, vertical writing where appropriate, and minimal chrome around the text.

03 · why

the official aozora site is a beautiful but dated archive. wanted something more legible for casual reading and study — and an excuse to learn how japanese typography works on the web.

04 · how

kotlin pipeline crawls aozora's index, downloads xhtml texts, normalizes encoding and ruby annotations. javascript frontend renders the catalog and reading view directly from the static output. no backend at runtime — just static files served wherever.
