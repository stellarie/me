yomitori

personal library for 40,000+ scattered ebooks.

2026 · typescript · kotlin

what

turns a folder of 40k+ ebooks into a searchable, beautiful library. crawls automatically, extracts covers and metadata, tracks reading progress, integrates with an epub reader. japanese tools layered on top — furigana rendering, frequency-based vocab, srs quizzes, anki sync.

why

i had a bunch of ebooks that became too unwieldy to browse. nothing was findable. wanted a real home — search with love, not find . -name.

how

typescript frontend, kotlin backend. docker compose orchestrates services. crawler runs on a schedule, persists to a local store. rest api drives search, pagination, and filtering.

headless cms

a prototype headless cms — visual editor + headless api.

2026 · spring boot · react

what

a drag-and-drop visual editor for composing pages, live preview before publishing, and a headless api to feed any frontend — react, vue, mobile. extensible: drop in your own components with custom groovy hooks.

why

most cmses force a tradeoff. headless ones are flexible but unfriendly to non-developers. visual builders trap you in their hosting. this prototype tries to keep both audiences happy — visual where it helps, api-first where it scales.

how

spring boot backend exposing the headless api and persisting page trees. react frontend for the visual editor with live preview. groovy hook layer lets developers register custom components without rebuilding the editor. ships as a static demo on github pages so the editor experience is browsable end-to-end.

aozora viewer

a reader for japan's public-domain literary archive.

2026 · javascript · kotlin

what

aozora bunko hosts thousands of out-of-copyright japanese literary works in raw xhtml. this project renders them with japanese typography, vertical writing where appropriate, minimal chrome around the text.

why

wanted something more legible for casual reading and study — and an excuse to learn how japanese typography works on the web.

how

kotlin pipeline downloads target xhtml text. javascript frontend renders catalog and reading view from the static output.

yoji jitenon scraper

a kotlin pipeline for harvesting four-character idioms.

2026 · kotlin

what

yojijukugo (四字熟語) are japan's classical four-character idioms — proverbs distilled into kanji. jitenon hosts the most complete free dictionary online. this scraper pulls every entry, with readings, meanings, sources, and usage notes, into a structured local dataset.

why

wanted offline access to the full corpus for personal study and for feeding into other projects (anki decks and other vocab tools). also a small excuse to write disciplined kotlin against real-world html.

how

plain kotlin with jsoup for parsing. polite crawl with rate limiting and on-disk caching so re-runs don't hammer the source. output is structured json — easy to consume from anything else downstream.
